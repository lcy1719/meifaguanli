## 【功能介绍】
**本系统包含管理员、用户、美容师三个角色。
管理员：用户管理、新闻公告管理、数据添加与维护、信息查询与修改、权限分配、安全验证、系统功能模块操作。
用户：登录系统、查询信息、提交留言、查看公告、收藏美容师与项目。
美容师：预约管理、项目管理、接收并回复留言、管理个人收藏。**
